# Vue.js Todo List (v1)

A simple Todo List web app built with Vue.js and Tailwind CSS

## Features
- เพิ่ม tasks
- เปลี่ยนสถานะ
- ลบ task
## Installation

1. **Clone the repository:**

 ```bash
 git clone https://github.com/nattapong2005/vuejs-todolist
```
2. **Install dependencies:**
```bash
npm install
```
3. **Start project:**
 ```bash
 npm run dev
```
